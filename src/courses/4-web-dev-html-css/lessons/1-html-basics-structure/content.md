## 1. What is HTML?
HTML (HyperText Markup Language) is the standard language used to create the structure of web pages.

## 2. Basic Structure of an HTML Document
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
</body>
</html>

## Practice Problems
### Easy
**Problem:** Create a simple HTML page with a heading and paragraph
**Solution:**

<!DOCTYPE html><html><body><h1>Welcome</h1><p>first page</p></body></html>

### Medium
**Problem:** Create a webpage that contains a heading, paragraph, clickable link, and image
**Solution:**

<!DOCTYPE html><html><body><h1>Webpage</h1><p>sample</p><a href="https://google.com">Google</a><img src="https://via.placeholder.com/150"></body></html>

### Hard
**Problem:** Create structured webpage with Header, Navigation, Main, Footer
**Solution:**

<!DOCTYPE html><html><body><header><h1>Header</h1></header><nav><a>Nav</a></nav><main><p>content</p></main><footer>Footer</footer></body></html>
