## 1. Lists
Unordered <ul>, ordered <ol>, description <dl>.

## 2. Tables
<table>, <tr>, <th>, <td>, colspan, rowspan.

## Practice Problems
### Easy
**Problem:** Unordered list of 3 fruits, ordered list of 3 steps
**Solution:**
 <ul><li>Apple</li></ul><ol><li>Step 1</li></ol>

### Medium
**Problem:** Table Name, Age, City with 3 rows
**Solution:**
 <table><tr><th>Name</th></tr><tr><td>Raja</td></tr></table>

### Hard
**Problem:** Nested list and table showing Product, Price, Availability with colspan
**Solution:**
 <ul><li>Electronics<ul><li>Phone</li></ul></li></ul><table><tr><th>Product</th><th colspan="2">Details</th></tr><tr><td>Phone</td><td>In Stock</td></tr></table>
