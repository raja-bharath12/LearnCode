## 1. What are Text Elements?
Text tags structure content readability.

## 2. Headings and Paragraphs
<h1> to <h6>, <p>, <br>

## 3. Formatting
<b>, <strong>, <i>, <em>

## 4. Semantic Tags
<header>, <nav>, <main>, <section>, <article>, <footer>

## Practice Problems
### Easy
**Problem:** Create a webpage with one h1, two paragraphs, line break
**Solution:**
 <h1>Welcome</h1><p>P1</p><br><p>P2</p>

### Medium
**Problem:** Heading and paragraph with bold, italic, highlighted text
**Solution:**
 <h1>Text</h1><p><b>bold</b> <i>italic</i> <mark>highlight</mark></p>

### Hard
**Problem:** Semantic page with header, nav, main, section, article, footer
**Solution:**
 <header><h1>Blog</h1></header><nav><a>Home</a></nav><main><section><h2>Posts</h2><article><p>Post</p></article></section></main><footer>©</footer>
