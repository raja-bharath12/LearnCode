## 1. Display
Block, inline, inline-block, none.
## 2. Position
Static, relative, absolute, fixed, sticky.

## Practice Problems
### Easy
**Problem:** block and inline element
**Solution:**
 <div style="display:block"></div><span style="display:inline"></span>

### Medium
**Problem:** Relative box moved 20 down right
**Solution:**
 position: relative; top: 20px; left: 20px;

### Hard
**Problem:** Absolute box in container with fixed navbar
**Solution:**
 nav{position:fixed;} container{position:relative;} box{position:absolute;}
