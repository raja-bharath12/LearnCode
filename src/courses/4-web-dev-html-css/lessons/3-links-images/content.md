## Practice Problems
### Easy
**Problem:** Link to Google and image displayed
**Solution:**
 <a href="https://google.com">Google</a><img src="image.jpg">

### Medium
**Problem:** Link in new tab, image custom size, clickable image link
**Solution:**
 <a href="https://google.com" target="_blank">Google</a><br><a href="https://wikipedia.org"><img src="image.jpg" width="100"></a>

### Hard
**Problem:** Anchor links jumping to sections with images
**Solution:**
 <nav><a href="#s1">S1</a></nav><h2 id="s1">S1</h2><img src="image.jpg">
