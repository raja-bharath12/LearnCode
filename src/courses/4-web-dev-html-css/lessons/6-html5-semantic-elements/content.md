## 1. Semantic Elements
<header>, <nav>, <main>, <section>, <article>, <footer>

## Practice Problems
### Easy
**Problem:** header, main, footer webpage
**Solution:**
 <header></header><main></main><footer></footer>

### Medium
**Problem:** header, nav, section
**Solution:**
 <header></header><nav></nav><section></section>

### Hard
**Problem:** Complete semantic structure
**Solution:**
 <header></header><nav></nav><main><section><article></article></section></main><footer></footer>
