## 1. Box Model
Margin, Border, Padding, Content. Built around CSS sizing and spacings.

## Practice Problems
### Easy
**Problem:** Box width 200 padding 10 border 2 black
**Solution:**
 .box{width:200px; padding:10px; border:2px solid black;}

### Medium
**Problem:** Margin 20, padding 15, background
**Solution:**
 .box{margin:20px; padding:15px; background:lightblue;}

### Hard
**Problem:** Default box sizing vs border-box
**Solution:**
 .box1{box-sizing:content-box;} .box2{box-sizing:border-box;}
