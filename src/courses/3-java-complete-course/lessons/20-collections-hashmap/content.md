## 1. HashMap
Key-value pairs.

## Practice Problems
### Easy
**Problem:** Add to HashMap.
**Solution:**
 `map.put("A", 1);`

### Medium
**Problem:** Count frequencies.
**Solution:**
 `map.put(key, map.getOrDefault(key, 0) + 1);`

### Hard
**Problem:** First non-repeating char.
**Solution:**
 Map char counts and check.
