## 1. Packages
Grouping classes. public, protected, private, default.

## Practice Problems
### Easy
**Problem:** Public vs Private.
**Solution:**
 Private is class-only. Public is universal.

### Medium
**Problem:** Protected.
**Solution:**
 Accessed in same package or subclasses.

### Hard
**Problem:** Create package.
**Solution:**
 `package com.example;`
