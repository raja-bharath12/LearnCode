## 1. Threads
Thread & Runnable.

## Practice Problems
### Easy
**Problem:** Thread class.
**Solution:**
 `class MyThread extends Thread { public void run() {} }`

### Medium
**Problem:** Runnable iterface.
**Solution:**
 `new Thread(new Runnable() { public void run(){} }).start();`

### Hard
**Problem:** Two threads printing numbers.
**Solution:**
 Start t1 and t2 together.
