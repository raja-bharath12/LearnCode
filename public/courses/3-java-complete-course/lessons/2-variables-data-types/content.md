## 1. What is a Variable?
A variable is a container used to store data in memory.
Types: Local, Instance, Static.

Primitive Data Types: byte, short, int, long, float, double, char, boolean.
Non-Primitive Data Types: String, Arrays, Classes, Objects.

## Practice Problems
### Easy
**Problem:** Declare variables and print them.
**Solution:**

```java
int age = 21; System.out.println(age);
```

### Medium
**Problem:** Type cast a double to int.
**Solution:**

```java
double d = 9.99;
int i = (int) d;
System.out.println(i);
```

### Hard
**Problem:** Convert a character 'A' to its ASCII value.
**Solution:**

```java
char c = 'A';
int ascii = (int) c;
System.out.println(ascii);
```
