## 1. Pseudos
:hover, :nth-child(), ::before, ::after

## Practice Problems
### Easy
**Problem:** Hover button
**Solution:**
 button:hover{background:blue;}

### Medium
**Problem:** first-child red, last blue
**Solution:**
 li:first-child{color:red;}

### Hard
**Problem:** ::before "Fire", ::first-letter big, hover blue
**Solution:**
 p::before{content:"🔥"} p::first-letter{color:red;}
