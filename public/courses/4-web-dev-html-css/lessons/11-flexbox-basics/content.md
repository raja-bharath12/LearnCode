## 1. Flexbox
1D flex layouts, row/col flex direction, justify-content, align-items.

## Practice Problems
### Easy
**Problem:** Row flex with 3 items
**Solution:**
 .container{display:flex;}

### Medium
**Problem:** Centered flex items with gap
**Solution:**
 .container{display:flex; justify-content:center; align-items:center; gap:20px;}

### Hard
**Problem:** 3 boxes, first takes more space
**Solution:**
 .b1{flex:2;} .b2{flex:1;}
