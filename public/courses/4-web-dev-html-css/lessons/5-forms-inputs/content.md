## 1. HTML Form
Collects user input using <form>, <input>, <textarea>, <button>, <select>

## Practice Problems
### Easy
**Problem:** Name, Email, Submit form
**Solution:**
 <form><input type="text"><input type="email"><input type="submit"></form>

### Medium
**Problem:** Name, password, radio genders, checkboxes skills, submit
**Solution:**
 <form><input type="password"><input type="radio" name="g"><input type="checkbox"></form>

### Hard
**Problem:** Complete registration form including select and textarea
**Solution:**
 <form><input type="text"><select><option>US</option></select><textarea></textarea></form>
