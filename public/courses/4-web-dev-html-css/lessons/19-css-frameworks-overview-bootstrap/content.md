## 1. Bootstrap
Ready CSS layout templates and classes like btn-primary, container.

## Practice Problems
### Easy
**Problem:** Bootstrap button
**Solution:**
 <button class="btn btn-primary">Me</button>

### Medium
**Problem:** container and two columns
**Solution:**
 <div class="container"><div class="row"><div class="col">1</div><div class="col">2</div></div></div>

### Hard
**Problem:** Navbar, btn, card
**Solution:**
 <nav class="navbar"></nav><button class="btn"></button><div class="card"></div>
