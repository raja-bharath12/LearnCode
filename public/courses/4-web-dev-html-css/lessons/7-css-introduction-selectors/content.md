## 1. What is CSS?
Styles web pages using inline, internal, external styles. Selector targeting.

## Practice Problems
### Easy
**Problem:** Inline blue color
**Solution:**
 <p style="color: blue;">Blue</p>

### Medium
**Problem:** Internal css h1 red, p green
**Solution:**
 <style>h1{color:red;} p{color:green;}</style>

### Hard
**Problem:** Class, ID, Group CSS
**Solution:**
 <style>.b{color:blue;} #r{color:red;} h2, p{font: Arial;}</style>
