## 1. Wrap and Align
Flex wrap, wrap-reverse, align-content, order, responsive flex flex:1 1 200px.

## Practice Problems
### Easy
**Problem:** Wrap items
**Solution:**
 display:flex; flex-wrap:wrap;

### Medium
**Problem:** Space rows with wrap
**Solution:**
 flex-wrap:wrap; align-content:space-between;

### Hard
**Problem:** Wrap, special item order -1 and self center
**Solution:**
 order:-1; align-self:center;
